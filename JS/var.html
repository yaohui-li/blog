    # var, let, const 关键字的区别

    ECMAScript 变量是松散类型的,即变量可以用于保存任务数据类型. 变量用于保存任意值的命名占位符. 声明变量有三个关键字: var、let、const. 其中 var 是任意 ECMAScript 版本都可以使用. let 和 const 只能在 ESCMAScript6 以及更晚的版本中可以使用

    1. # var

      var 的使用很简单,如下例子:

    var name; // 声明变量但是不赋值, 此时name变量为undefined
console.log(name); // undefined
var message = "my name is mike"; // 声明变量并将'my name is mike'字符串赋值给变量message
// 初始化多个变量
var age = 37,
  job = "painter";

function getUser() {
  var user;
  console.log(user); // undefined 而不是报错

    1. # const

    const 和 let 基本是相同的, 区别在 const 声明变量必须同时初始化变量,且不能修改 const 声明的变量值. 例子如下:

     

     

    const name; // Missing initializer in const declaration
const age = 37;
age = 32; // TypeError Assignment to constant variable.
    1
    2
    3

    由于 const 只限制变量指向变量的引用,假如变量是一个对象,修改其对象内部属性或者方法不会违反 const 的限制.即不会报错

    const user = {
  name: "mike",
  age: 34,
};
user.name = "jodan";
console.log(user.name); // jodan

    建议

    1. 不使用 var
    2. 声明变量优先级 const > let

    总结: var, let, const 区别

    关键字 作用域 相同变量重复声明 变量提升 是否可以修改
    var 函数作用域 可以 可以 可以
    let 块级作用域 不可以 不可以 可以
    const 块级作用域 不可以 不可以 基本数据类型不可以,
    对象的引用不能改但内部属性或方法可以
