    # JavaScript 的数据类型之原始值

    目前 JavaScript 的数据数据类型分为原始数据类型和引用数据类型. 原始数据类型分为: Undefined, null, Boolean, Number, String, Symbol, BigInt; 引用数据类型为 Object(对象). Object 是一种无序键值对的集合

    # 原始数据

    • Undefined

      Undefined 含义是变量声明但是未赋值, 例如:

      let message; // 此时变量声明 但是没有赋值 默认为undefined
console.log(message); // undefined
console.log(age); // 报错 因为没有声明age这个变量

    • Null

      Null 表示一个空对象指针. undefined 值是由 null 值派生出来的. 因此在表面上它们是相等的.

      console.log(null == undefined); // true

    • Boolean

      Boolean 只有两个值 true 和 false.

    • Number

      Number 用来表示整数和浮点值,可以表示不同进制的数据

      • 整数
      let intNum = 34; // 34 十进制
let octalNum = 070; // 八进制 56
let hexNum = 0xa; // 十六进制 10

      • 浮点数

        ECMAScript 采用 IEEE754 双精度标准.

      console.log(0.1 + 0.3); // 0.4
let fNumber = 0.4; // 不推荐写法
console.log(0.1 + 0.2); // 0.30000000000000004   ???

      思考

      💡 0.1 + 0.2 为什么不等于 0.3 呢 ECMAScript 是按照 IEEE754 64 位双精度标准. 当十进制换算成二进制时,由于 64bit 空间存储不足, 通过 IEEE754 来舍入后计算再转换成十进制从而得出 0.30000000000000004

      💡 为什么 isNaN({})为 false,而 isNaN([])却是 true❔

      因为 isNaN 方法内部首先对对象/数组调用 valueof()后检查是否能转为数字. 如果不能, 再调用 toString()方法, 检查是否能转为数值.

      isNaN([]); // false
console.log([].valueOf()); // []
console.log([].valueOf().toString()); // ""
console.log(Number("")); // 0
// 因此isNaN([]) 为 false
let obj = {};
isNaN(obj); // true
console.log(obj.valueOf()); // {}
console.log(obj.valueOf().toString()); // "[object Object]"
// 因此isNaN(obj) 为 true

    • String String(字符串)表示零个或多个 16 位 Unicode 字符序列. 可以用"" '' ``来标示.

    • Symbol

      Symbol 是 ECMAScript6 新增数据类型, 符号是原始值且符号实例是唯一、不可变的.主要用途确保对象属性使用唯一标识符

      • 基本用法
      const symbol = Symbol();
console.log(typeof symbol); // symbol

const sym = Symbol("key");
const otherSym = Symbol("key");
console.log(sym === otherSym); // false

    • BigInt

      BigInt 是一种来表示 253 - 1 的整数. 它可以表示任意大的整数 具体介绍 (opens new window)

      console.log(typeof 1n); // 'bigint'
console.log(typeof BigInt(2n) === "bigint"); // true

console.log(1n + 2n); // 3n
console.log(3n / 2n); // 1n, not 1.5n 会取整

console.log(1n === 1); // false
console.log(1n == 1); // true
