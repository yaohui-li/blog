    # JavaScript 的数据类型之引用值

    在 JavaScript 中数据类型主要分为原始值和引用值. 在 JavaScript 引擎中原始值的变量是按值访问,存放在栈中. 而引用值的变量按引用访问值. 引用地址存入栈中, 值存放在堆里. 引用类型统称为 Object 类型, 但具体划分的话, 分为 Object, Date, Math, RegExp, Function, Array

    • 声明引用变量

      const obj = { name: "Nicholas" }; // 对象字面量声明
const user = new Object({ name: "Tom", age: 36 }); // 构造函数声明
const date = new Date(); // 日期对象
const reg = new RegExp("/d+/"); // 正则对象
const PI = Math.PI; // 数学圆周率
// 声明函数
function getUser() {
  return {
    name: "Mike",
    job: "writer",
  };
}

    • 原始值和引用值之间差异

      • 存储方式

        原始值存储于栈中, 引用值存储于堆中而引用地址存于栈中

      • 变量复制

        原始值是值得拷贝, 引用值则是引用地址的拷贝

      • 参数传递

        原始值和引用值作为参数都是值的传递.

          const age = 35;
  const user = {
    name: 'liyaohui'
  }
  function addAge(age) {
    return age + 1;
  }
  function setUser(user) {
    user.name = 'xiehanbing';
    user = new Object();
    user.name = 'Jordan';
  }

  // age的值拷贝给形参age
  // 函数中的age 此时是一个局部变量
  console.log(addAge(age)); // 36
  console.log(age);
  // user引用值拷贝一个引用相同的引用值给到形参user
  // 即便形参user被赋值一个新的引用值 但是不会改变之前的引用值的数据 函数setUser运行完 形参user便会销毁~
  console.log(user.name); // xiehanbig
